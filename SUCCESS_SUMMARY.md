## 🚀 立即可用的安装流程

### 快速启动 (5分钟)
```bash
# 1. 安装依赖 (已验证成功)
npm install

# 2. 构建项目 (已验证成功)
npm run build

# 3. 启动开发服务器
npm run dev-server

# 4. 在PowerPoint中加载
npm run sideload
```

### 手动安装
1. 打开PowerPoint
2. 文件 → 选项 → 信任中心 → 信任中心设置
3. 受信任的加载项目录 → 添加 `E:\code\FreeCut\dist` 文件夹
4. 重启PowerPoint
5. 插入 → 我的加载项 → FreeCut
